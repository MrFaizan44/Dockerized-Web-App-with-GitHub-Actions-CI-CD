# DevOps Demo

## About
A tiny web app to demonstrate Docker and GitHub Actions CI.

## Tech Stack
- HTML, CSS
- Docker
- GitHub Actions

## How to run
1. Build Docker image: `docker build -t faizan-demo .`
2. Run container: `docker run -p 8080:80 faizan-demo`
3. Visit: `http://localhost:8080`

## Skills
- Containerization
- CI/CD pipeline (GitHub Actions)
- Basic Web Development
